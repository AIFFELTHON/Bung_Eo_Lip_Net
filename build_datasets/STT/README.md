## 준비물
- videos.json
- `youtube` 영상 링크를 모아둔 `Json` 파일

```json
// -ex

{
	"파일이름":"링크",
	"002":"https://www.youtube.com/watch?v=bQF4umz6YCE&t=11s"

}
```

## 사용법

```py
python3 auto.py

```

- ```videos.json```에 다운받을 영상 목록을 작성한후 ```auto.py```를 실행
- 영상을 GCS에 옮겨담고 STT추출
- STT json파일은 ```wavs/```폴더 안에 담김
